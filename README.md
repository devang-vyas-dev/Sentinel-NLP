Sentinel-NLP is a Python-based automated content moderation system designed to identify and categorize harmful language in digital communications. Developed as the foundational phase of a larger Natural Language Processing (NLP) roadmap, Version 1.0 utilizes a robust rule-based engine to scan strings for toxic, racist, vulgar, and exclusionary language.

By mapping user input against categorized linguistic datasets, the system provides real-time verdicts and educational feedback to promote safer, more inclusive online environments. This project serves as a structural precursor to future iterations involving Sentiment Analysis and Contextual Machine Learning models.

Key Technical Features:

Categorized Lexicon Mapping: Organized data structures for Toxic, Racist, Vulgar, and Body Shaming classifications.

Case-Insensitive Normalization: Pre-processing logic to ensure consistent detection regardless of user formatting.

Modular Architecture: Clean, function-based design allowing for easy integration into larger Python frameworks.

Instant Feedback Loop: Automated verdict generation with context-specific ethical warnings.
